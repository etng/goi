## 为什么用 Goi

如果你手上有一堆下载来的 MDX 词典，平时查词却要打开笨重的老软件——Goi 想让这件事
变得轻快：常驻在菜单栏，一个快捷键就查，查完就走；而你查过的每一个词都不会白查，
它们会慢慢积累成你自己的生词本和学习记录。

Goi 完全在你的电脑本地运行，不联网、不上传，你的词典和学习数据都只在你自己手里。
