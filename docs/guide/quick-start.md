# 快速上手

装好 Goi、[导入了词典](./add-dictionaries)之后，日常就这么用：

## 查一个词

按 **⌥ Space**（Option + 空格），查词窗口就会弹出，光标已经在输入框里。输入要查的词，
边打边出结果。按 **Esc** 关闭窗口。

![查词窗口](/shots/lookup-en.png)

就这么简单。下面这些你会自然而然用到：

- 上方一排是**各本词典的标签**，点一下只看那本；「全部」看所有。没收录这个词的词典是灰的。
- 输入变形词（`went`、`studies`）会**自动按原型查到**，并在顶部提示。
- 在别的软件里选中文字，按 **⌃⌥ Space** 可以[直接查选中的词](./selection)。

## 常用快捷键

| 快捷键 | 作用 |
| --- | --- |
| ⌥ Space | 打开／关闭查词窗口 |
| ⌃⌥ Space | 查询当前选中的文字 |
| Esc | 关闭窗口 |
| 回车 | 查询输入框里的词 |
| ⌘↩ | 在心得框里保存心得 |

两个全局快捷键（打开面板、划词）都可以在**设置 → 快捷键**里自己改。

## 窗口的一些小技巧

- 双击标题栏可以最大化／还原窗口。
- 拖动边缘可以调整大小，Goi 会记住你调好的尺寸。
- 打开窗口时可以用 ⌘Tab 切到它；关掉后 Goi 退回菜单栏，不占程序坞。
