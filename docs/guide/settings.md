# 设置

点左侧的**设置**，这里集中了几样常用配置。

![设置页](/shots/settings.png)

## 词典库

- 「**添加词典…**」导入新词典（见[添加词典](./add-dictionaries)）。
- 「**重新扫描**」在你手动改动过词典库后刷新。

## 外观

三种模式随你切换：

- **跟随系统** — 跟随 Mac 的浅色/深色设置。
- **浅色** — 始终浅色。
- **深色** — 深色界面，词典内容仍是清晰的白底卡片，晚上看更舒服。

## 依赖

这里显示两个可选组件是否就绪，装好后点「重新检测」：

- **mecab** — 日语变形还原。没装时可点「复制安装命令」，到终端粘贴运行。
- **AnkiConnect** — 与 Anki 的连接，可点「检测连接」确认。

（安装方法见[安装](./install#让日语和发音更完整-可选)。）

## 词典顺序

- **拖动**调整词典先后（也就是查词页标签的顺序）。
- **双击**词典名可起个简短**别名**，方便在标签上辨认。
- 每本词典显示**用过多少次**和词条数，帮你判断取舍；对着词典点右键可以「从库中移除」。
