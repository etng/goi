# 添加词典

Goi 本身不带词典，第一次使用要先把你自己的词典导入进来。

## 导入步骤

1. 点菜单栏的「語」图标打开窗口，点左侧的**设置**。
2. 在「词典库」一栏点「**添加词典…**」。
3. 选中你的 `.mdx` 词典文件，或直接选中**装着一堆词典的文件夹**（比如你放词典的
   `dicts` 目录），Goi 会自动找出里面所有词典一次性导入。
4. 稍等片刻，导入完成后就能查词了。

支持的文件：`.mdx`（词典正文）和配套的 `.mdd`（图片、发音等）。同名的 CSS、图片会一起导入，保证排版正常。

![设置里的词典库与词典管理](/shots/settings.png)

## 导入不会占用额外空间

Goi 导入词典用的是 macOS 的「克隆」技术：在同一块磁盘上，导入几个 G 的词典
**几乎不占额外空间、瞬间完成**。

更省心的是——**导入之后，你原来的词典文件就可以随意移动或删除了**，完全不影响
Goi 使用。Goi 用的是自己库里的副本。

## 想删掉某本词典

在设置的词典列表里，对着某本词典点**右键 →「从库中移除」**。这只会删掉 Goi 库里的
那份副本（还会放进废纸篓，可以找回），你自己的原始词典文件不受任何影响。

## 整理与排序

- **拖动**列表里的词典可以调整顺序，顺序就是优先级：决定查词页标签的排列、
  「全部」视图里谁在前面。
- **双击**词典名字可以给它起个**简短的别名**（比如把很长的名字改成「牛津 9」），
  方便在标签上辨认。
- 每本词典右侧显示你**用过它多少次**，可以据此判断哪些词典其实不常用。详见[历史与统计](./history-stats)。
