# 查词与切换词典

## 一次查所有词典

输入一个词，Goi 会同时在你所有词典里查，结果按词典分成一节一节显示。

顶部一排是**词典标签**：
- 点某个词典名，就只看那一本的解释。
- 点「**全部**」看所有词典。
- 某本词典没收录这个词时，它的标签是**灰色**、点不动。
- 每个标签后面的小数字是这本词典里的**词条数**。

![多本词典的标签](/shots/dark.png)

### 调整词典顺序、设默认

- 直接在标签上**拖动**就能调整词典的先后顺序。
- 对着某个标签点**右键 →「设为默认词典」**，以后查词就默认先显示这一本（标签上会有个图钉标记）。没设默认时，Goi 会自动选中第一个有结果的词典。

## 变形会自动还原

不确定原型也没关系，直接输入你看到的形式：

- 英语：输入 `went` 会查到 `go`，`studies` 查到 `study`，并在顶部提示「已按原型 … 查询」。
- 日语：输入「食べました」会还原成「食べる」。（这一项需要安装 mecab，见[安装](./install#让日语和发音更完整-可选)。）

![按原型查询的提示](/shots/lemma.png)

## 听发音、看图片

词典里带的发音喇叭、图片都能正常显示和播放，点一下喇叭就能听。

部分词典（如 NHK 发音辞典）的语音是一种特殊格式，需要安装 `ffmpeg` 才能播放；
没装时点了会提示你。

## 跟着链接跳转

解释里带下划线的词是**跳转链接**，点一下就查那个词，搜索框会显示你点的词。
