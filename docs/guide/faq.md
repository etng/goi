# 常见问题

## 打不开，提示「无法验证开发者」

这是因为 Goi 没有购买苹果的付费签名，不是安全问题。在 `Goi.app` 上点**右键 →
「打开」**，或到「系统设置 → 隐私与安全性」点「仍要打开」。只需一次。详见[安装](./install#首次打开被拦住)。

## 按 ⌥ Space 没反应

- 确认菜单栏上有「語」图标（说明 Goi 在运行）。
- 这个快捷键可能被别的软件占用了。可以先点菜单栏的「語」图标 →「打开查词面板」确认能打开。

## 划词（⌘⌥ Space）查不到选中的词

需要先授权：「系统设置 → 隐私与安全性 → 辅助功能」里打开 **Goi**。详见[划词取词](./selection)。

## 日语变形查不到（如「食べました」）

需要安装 mecab：终端运行 `brew install mecab mecab-ipadic`，再到设置里点「重新检测」。
查原型（如「食べる」）不受影响。

## 点发音喇叭没声音

多数词典的发音可以直接播放。少数词典（如 NHK 发音辞典）用的是特殊格式，需要
安装 `ffmpeg`：终端运行 `brew install ffmpeg`。

## 删掉原来的词典文件，Goi 还能用吗

能。导入时 Goi 已经在自己的库里存了一份副本，你的原始文件可以随意移动或删除。
想真正释放空间，要在 Goi 的设置里对词典点右键「从库中移除」。详见[添加词典](./add-dictionaries#导入不会占用额外空间)。

## 我的数据会上传吗

不会。Goi 完全在本地运行，词典和学习记录都只保存在你自己的电脑上。只有你主动
点「同步到 Anki」时，生词才会发送到你本机运行的 Anki。

## 换了新电脑，怎么把生词带过去

在生词本点「导出 JSON」存成文件，在新电脑上用「导入 JSON」导入即可，包含历史和心得。

## 在哪里反馈问题

到 [GitHub Issues](https://github.com/etng/goi/issues) 提交。
