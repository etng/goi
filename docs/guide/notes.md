# 写下你的心得

查词不只是看解释。Goi 在每个词的查词页**底部固定**留了一块「我的心得」，
让你随手记下自己的理解、联想、好的例句或记忆方法。

## 怎么用

1. 查到一个词，看向窗口底部的「我的心得」。
2. 在输入框里写点什么，按「保存」或 **⌘↩**。
3. 你写的心得会按时间**倒序**列在下面，下次查到这个词时一并显示。

心得会跟着这个词一直保存。想收起这块区域，点它右边的箭头即可。

## 为什么值得写

看懂一个词的解释，和真正把它变成自己的，中间差的往往就是一次"用自己的话说一遍"。
把当时的理解、联想或第一次遇到它的句子记下来，比反复看解释更容易记住。

::: tip
这些心得也会随生词本一起[导出](./anki#导出成文件)，不会丢。
:::
