# 划词取词

在浏览器、PDF、聊天软件里读到不认识的词，不用手动敲进去——选中它，按一下快捷键就查。

## 怎么用

1. 在任意软件里用鼠标**选中一个词**。
2. 按 **⌘⌥ Space**（Command + Option + 空格）。
3. Goi 弹出并显示这个词的解释。

Goi 还会顺手记住这个词**所在的整句话**，作为例句留在生词本里，以后做 Anki 卡片时能直接用上。

## 第一次要授权

这个功能需要读取你在别的软件里选中的文字，所以要开一个系统权限：

打开「**系统设置 → 隐私与安全性 → 辅助功能**」，把 **Goi** 的开关打开，然后再按一次 ⌘⌥ Space。

只需授权一次。如果你之前没在[安装](./install)时开，现在补开即可。

::: tip
不想用划词也完全没问题，普通的 ⌥ Space 查词不需要任何权限。
:::
