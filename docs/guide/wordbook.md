# 生词本与熟悉度

你查过的词不会白查。Goi 会帮你把它们攒成一个**生词本**，并记住你对每个词的
**熟悉程度**。

## 生词本怎么来的

- **自动加入**：同一个词查到第二次，它就自动进入生词本——你反复查的，往往正是记不牢的。
- **手动加入**：查词时点搜索框右边的 **☆**，就把当前的词加入生词本，并给它更高的权重。

点左侧的**生词本**查看全部。点任意一个词可以直接跳到它的解释（这种"回看"不会增加它的查询次数）。

![生词本](/shots/wordbook.png)

## 熟悉度是什么

每个词都有一个 0–100 的**熟悉度**，数字越低表示你越不熟：

- 新词起始较高，**每查一次就降一点**（越查说明越不熟）。
- 一段时间不再查，它会**慢慢回升**（大概是你已经记住、或不再需要了）。
- **手动加入**的词熟悉度会明显降低，并且不再自动回升——因为这是你主动标记的重点。

生词本里用红/橙/绿的小数字直观表示：红=最陌生，绿=已比较熟。你可以据此决定先复习谁。

## 管理

- 每个词右侧的垃圾桶图标可以把它移出生词本。
- 想把这些词拿去复习或备份，见[同步到 Anki](./anki)。
