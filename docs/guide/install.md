# 安装

## 下载并安装

1. 打开[最新发布页面](https://github.com/etng/goi/releases/latest)，下载 **`Goi.dmg`**（通用版，Apple 芯片和 Intel 都能用）。
2. 双击打开，把 **Goi 拖进 Applications（应用程序）**文件夹。
3. 到「应用程序」里双击 Goi 打开。屏幕顶部菜单栏出现「語」图标，就说明它已经在运行了。

## 首次打开被拦住？

Goi 当前发布版尚未完成 Apple Developer 签名和公证，所以第一次打开时系统会提示
「**无法验证**」或「**已损坏**」——这不是病毒，而是 macOS 对未公证软件的例行拦截。
请先确认安装包来自 Goi 官方发布页。最直接的处理方式是在「终端」运行：

```sh
xattr -dr com.apple.quarantine "/Applications/Goi.app" && open "/Applications/Goi.app"
```

这条命令只移除当前 Goi 副本的下载隔离标记，不会关闭 Mac 对其他应用的安全检查。
如果提示没有权限，在 `xattr` 前加 `sudo`。新下载的版本如果再次被拦截，需要对新的 App
再执行一次。

不使用终端时，也可以按下面三步从系统设置放行：

**第一步**：第一次双击会看到这个提示，点「完成」关掉它（**不要**点「移到废纸篓」）。

![无法验证的提示](/shots/gatekeeper-1-blocked.png)

**第二步**：打开「**系统设置 → 隐私与安全性**」，往下找到「已阻止 Goi.app…」，
点右边的「**仍要打开**」。

![在隐私与安全性里点仍要打开](/shots/gatekeeper-2-settings.png)

**第三步**：在弹出的确认框里再点一次「**仍要打开**」。

![确认仍要打开](/shots/gatekeeper-3-confirm.png)

搞定，当前安装的 Goi 就可以打开了。

## 让「划词取词」可用（可选）

如果你想在别的软件里选中文字直接查词，需要给 Goi 一个系统权限：

打开「**系统设置 → 隐私与安全性 → 辅助功能**」，把 **Goi** 的开关打开。

没开也不影响普通查词，只是选中文字查词这个功能用不了。详见[划词取词](./selection)。

## 让日语和发音更完整（可选）

这两项不装也能用大部分功能，需要时再装：

- **日语变形还原**（把「食べました」还原成「食べる」）：在「终端」里运行
  `brew install mecab mecab-ipadic`，然后到 Goi 设置里点「重新检测」。
- **部分词典的发音**（如 NHK 发音辞典的语音）：运行 `brew install ffmpeg`。

装好后无需重启，Goi 会自动识别。
