# 安装

## 下载并安装

1. 打开[最新发布页面](https://github.com/etng/goi/releases/latest)，下载 `Goi.zip`。
2. 双击解压，把 `Goi.app` 拖进「应用程序」文件夹。
3. 双击打开。屏幕顶部菜单栏会出现一个「語」图标，就说明 Goi 已经在运行了。

## 首次打开被拦住？

Goi 是免费开源软件，没有购买苹果的付费签名，所以第一次打开时系统可能提示
「无法验证开发者」。这不是病毒，按下面任一种方式打开即可：

- **右键打开**：在 `Goi.app` 上点右键 → 「打开」→ 在弹窗里再点一次「打开」。
- 或到「**系统设置 → 隐私与安全性**」，在最下方找到关于 Goi 的提示，点「仍要打开」。

只需这样做一次，之后正常双击即可。

## 让「划词取词」可用（可选）

如果你想在别的软件里选中文字直接查词，需要给 Goi 一个系统权限：

打开「**系统设置 → 隐私与安全性 → 辅助功能**」，把 **Goi** 的开关打开。

没开也不影响普通查词，只是选中文字查词这个功能用不了。详见[划词取词](./selection)。

## 让日语和发音更完整（可选）

这两项不装也能用大部分功能，需要时再装：

- **日语变形还原**（把「食べました」还原成「食べる」）：在「终端」里运行
  `brew install mecab mecab-ipadic`，然后到 Goi 设置里点「重新检测」。
- **部分词典的发音**（如 NHK 发音辞典的语音）：运行 `brew install ffmpeg`。

装好后无需重启，Goi 会自动识别。
