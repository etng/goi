# Goi 是什么

Goi（語彙）是一款 macOS 上的**本地词典应用**。你把自己收藏的词典文件导入进来，
之后无论在看网页、读 PDF 还是写东西，按一个快捷键就能立刻查词。

它常驻在屏幕顶部的菜单栏里，平时不打扰你；需要时随叫随到，查完一按 Esc 就消失。

## 它能帮你做什么

- **一处查所有词典**：不用一本本翻，输入一个词，所有词典的解释一起出来，可以按词典分栏切换。
- **看不懂的变形也能查**：输入 `went`、`studies` 这类变化过的词，会自动按原型查到；日语的活用（如「食べました」）也能还原成「食べる」。
- **查过的词帮你记着**：反复查的词会自动进入生词本，并记录你对它的熟悉程度——越是记不住、查得越多的词，排得越靠前。
- **记录自己的心得**：每个词都能写下你的理解、联想或例句，下次查到时一并显示。
- **和 Anki 打通**：把生词一键做成 Anki 卡片，在手机上复习；复习情况还能反过来更新这里的熟悉度。

## 它需要什么

- 一台 **macOS 13 或更新版本**的 Mac。
- 你自己的词典文件（`.mdx`，可选配套的 `.mdd`）。Goi 本身不附带任何词典。

::: tip 关于词典文件
Goi 不提供词典内容，你需要自备合法获取的 MDX 词典。导入后所有数据都只保存在
你本地。
:::

## 和常见词典软件对比

下面拿 Goi 和两款常见的、同样能加载自有 MDX 词典的软件做个对照：**GoldenDict-ng**
（开源，很多人用来查 MDX 词典）和**欧路词典**（Eudic，功能全面的商业软件）。

| 能力 | GoldenDict-ng | 欧路词典 | Goi |
| --- | :---: | :---: | :---: |
| 平台 | Windows / Linux / macOS | 全平台 + 手机 | **仅 macOS** |
| 开源免费 | ✅ 开源 | 免费 + 内购 | ✅ 开源 |
| 加载自有 MDX / MDD 词典 | ✅ | ✅ | ✅ |
| 一次查多本词典、可切换 | ✅ | ✅ | ✅ |
| 划词取词 | ✅ | ✅ | ✅ |
| 发音 | ✅ | ✅ | ✅ |
| 词形还原（变形查原型） | ✅ | ✅ | ✅ 英语 / 日语 |
| 生词本 | 仅收藏夹 | ✅ 核心功能 | ✅ |
| **导入不占额外空间、原文件可随意删** | — | — | ✅ 独有 |
| **按查词行为自动记录熟悉度** | — | 需手动分级 | ✅ 自动 |
| **每个词写个人心得** | — | ✅ 笔记 | ✅ 固定在查词页 |
| **词典使用率统计 + 取舍建议** | — | — | ✅ 独有 |
| 与 Anki 打通 | 需借助第三方 | 导出为主 | ✅ 双向（AnkiConnect） |
| 纯本地、不联网 | ✅ | 云功能需联网 | ✅ 纯本地 |
| 界面 | 功能强、偏传统 | 现代、商业化 | 原生轻量、菜单栏 + 快捷键 |

::: tip 加粗的几行是我们特别用心的地方
- **导入用 APFS 克隆**：几个 G 的词典瞬间导入、几乎不占空间，导入后原文件随便删。
- **熟悉度自动记录**：不用手动给每个词分级，你查得多的词自然浮上来。
- **词典使用率统计**：帮你看清哪些词典常用、哪些可以清理，别人没有这个。
- **Anki 双向同步**：不只是导出，复习结果还能回来更新熟悉度。
:::

我们也坦白**目前还不如它们的地方**：Goi 只支持 macOS，没有手机端和云同步，也还没有
全文检索（在词典正文里搜句子）——这些在后续计划中。如果你需要跨平台或海量在线词典，
它们仍是很好的选择；如果你主要在 Mac 上用自己的 MDX 词典、看重积累和复习，Goi 会很趁手。
