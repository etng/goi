# 历史与统计

## 查询历史

点左侧的**历史**，能看到你查过的每一个词，按时间倒序、分页显示，包括这个词是
键入查的、点候选查的还是点链接查的。点任意一条可以再看一次它的解释。

## 统计

点左侧的**统计**，看两样东西：

![统计页](/shots/stats.png)

### 查词热力图

像 GitHub 那样的方格图，颜色越深表示那天查得越多。下面有今天、近 7 天、累计的次数。
一眼看出自己最近学得勤不勤。

### 词典效率

这张表帮你判断**哪些词典值得留、哪些可以清理**：

- **命中率**：你查过的词里，这本词典收录了多大比例。
- **用过 N 次**：你**主动点开**这本词典标签的次数（只是自动一起查出来的不算）。
- 右侧的标签是结论：
  - **主力词典** — 又收得全、你又常看，留着。
  - **查得到但没在看** — 收得全但你从不点开，可以往后排。
  - **可考虑移除** — 既查不到又不用，是清理磁盘空间的首选。

点右上角「**按使用频率重排词典顺序**」，可以一键把你最常用的词典排到最前。

::: tip
熟悉度的打分规则日后可能微调；因为所有查询历史都完整保留着，调整后可以对
历史重新计算，你的数据不会丢。
:::
