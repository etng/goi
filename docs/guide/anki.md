# 同步到 Anki

[Anki](https://apps.ankiweb.net/) 是一款流行的免费记忆卡片软件。Goi 可以把你的
生词做成 Anki 卡片，让你在手机、平板上随时复习。

## 准备工作

1. 电脑上装好 Anki 并打开它。
2. 在 Anki 里安装 **AnkiConnect** 插件（Anki 菜单：工具 → 附加组件 → 获取附加组件，
   输入代码 `2055492159`，重启 Anki）。

装好后让 Anki 保持开着即可。

## 推送生词

在 Goi 的**生词本**页，点「**同步到 Anki**」。Goi 会：

- 自动在 Anki 里建一个叫 **Goi** 的牌组和一种叫 **Goi Word** 的卡片模板；
- 把生词做成卡片，带上词头、释义、熟悉度、查询次数等信息；
- 已经推送过的词再次同步时只更新，不会重复。

之后就能在 Anki（包括手机 App）里复习这些词，并随 AnkiWeb 同步到所有设备。

## 把复习结果带回来

在生词本点「**从 Anki 回读**」，Goi 会读取你在 Anki 里的复习情况：**记得越牢
（间隔越长）的词，这里的熟悉度会相应调高**。这样两边的进度就对得上了。

## 导出成文件

不想用 Anki，也可以直接导出：

- **导出 CSV**：生词简表，可用 Excel 打开，也能直接导入 Anki。
- **导出 JSON**：包含查询历史和心得在内的完整备份，可用「导入 JSON」还原或迁移到另一台电脑。

::: tip
建议偶尔导出一次 JSON 备份——生词本是你长期积累的成果，值得存一份。
:::
